# AI Study Assistant 🤖

AI Study Assistant is an AI-powered learning application that helps students understand concepts, ask questions, and get instant AI-generated explanations.

## Features

- AI-powered question answering
- Chat-based learning interface
- FastAPI backend
- Gemini AI integration
- Secure API key management using environment variables
- Dockerized backend deployment
- Responsive frontend UI

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Python
- FastAPI
- Google Gemini API

### Deployment Tools
- Docker
- Git
